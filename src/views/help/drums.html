Drum and percussion editor is designed to be as simple as possible. All sounds are described by single property - volume intensity. Sound length (duration) depends only on particular drum type (sound sample). Currently there is no option to untimely mute longer lasting sounds, but it is planned to be implemented.

 


Basic sound editing controls:

  • Click once to create loud drum sound
  • Click again to lower drum sound
  • Click again to clear drum sound
  • Use mouse wheel for more precise volume level control:
    • Scroll-up direction to raise volume
    • Scroll-down direction to lower volume


Note:   Don't forget on functions available in Barline context menu, like Copy, Paste, Clear or beat subdivision options.

Drag & Drop operation can be used to:

  • Move existing sound to the new position (same drum type sound)
  • Copy existing sound to the new position - hold Ctrl key during dragging
