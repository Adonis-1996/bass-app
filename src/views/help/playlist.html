Playlist is an ordered list of sections, where each section can be repeated multiple times. Typically, every project should have at least one playlist for the whole composition, but when your project is composed of many small section, it may be useful to create playlists for bigger logical blocks like verses or choruses.

Note:   Use can use Drag & Drop operation in Playlists selectbox to change order of project's playlists.


To open Playlist editor, you have to be in Playlist Mode and then click on .
Playlist editor consists of two lists:

  • Playlist - actual playlist
  • Section - list of project's sections, but only the ones not already included in the current playlist (for easier tracking of changes)

Edit functions:

  • Drag items (sections) from Section list into the Playlist
  • Drag items inside Playlist list to change order, or duplicate sections in combination with pressed Ctrl key
  • Enter number of repeats of particular section in text field (visible when hovering item)
  • Select and delete item in Playlist with Delete key
  • Use Clear button to remove all sections from current playlist
  • Use Move all button to move all project's sections into the playlist
