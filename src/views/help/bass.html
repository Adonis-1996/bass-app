Sound Properties menu contains all properties used to describe individual bass sounds, like playing style, note (type, pitch, length, ...) and volume. It also contains a playback function for easy check of resulting sound. You can open it by:

  • click on
    +
    button in bass editor area to create a new sound at specified string and time (subbeat)
  • right click on existing bass sound element

 

 

 

 

 

Event though this menu allows to edit all sounds properties, application offers additional ways to create or edit sounds more effectively. Next topics will cover this methods.

Drag & Drop operation can be used to:

  • Create a new sound by dragging particular note from fretboard
  • Move existing sound to the new position (string and time)
  • Copy existing sound to the new position - hold Ctrl key during dragging

{{ ::item.sections[2] }}

Resize operation is executed by dragging resize handler - right side of the sound element. It can be used to:

  • Change note length of the sound
  • Combine two sounds on the same string (regular notes with different pitch) into the slide sound

Steps:

    Resize operation:
    • 1. Click and hold left mouse button on the right edge of the sound to start resize operation
    • 2. Resize sound to the desired note length

    Create slide:
    • 1. Click and hold left mouse button on the right edge of the first sound to start resize operation
    • 2. Resize first sound to overlap with the second sound

Some playling techniques like Hammer-On and Pull-Off require smooth transition from one note into the other one. On the bass guitar it means playing two or more notes on single string pluck/stroke. Another reason to connect multiple sounds is a Tie notation, where two or more notes are joined and played together with the time values summed.

Steps:

  • 1. Create a first sound
  • 2. Create a second sound on the same string (right after the first sound)
  • 3. Select a second sound
  • 4. Press keyboard shortcut for particular sound transition type

 

Keyboard shortcuts:

  • Ctrl + H - Hammer-On
  • Ctrl + P - Pull-Off
  • Ctrl + R - Let ring (Tie when both sounds have the same pitch, slide transition otherwise)
