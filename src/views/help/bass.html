Sound Properties menu contains options for creation and adjustment of individual bass sounds, like playing style, note (type, pitch, length, ...) and volume. It also contains a playback function for easy check of resulting sound.

Steps:

  • 1. Create a first sound
  • 2. Create a second sound on the same string (right after the first sound)
  • 3. Select a second sound
  • 4. Press keyboard shortcut for particular sound transition type

 

Keyboard shortcuts:

  • Ctrl + H - Hammer-On
  • Ctrl + P - Pull-Off
  • Ctrl + R - Let ring

Steps:

  • 1. Create first sound
  • 2. Create second sound on the same string
  • 3. Click and hold left mouse button on the right edge of the first sound to start resize operation
  • 4. Resize first sound to overlap with the second sound
