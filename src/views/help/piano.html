  • Delete - delete selected sound
  • Up - raise note pitch by one semitone (fret)
  • Down - lower note pitch by one semitone (fret)
  • Tab - toggle sharp/flat note name
  • . - toggle staccato attribute
  • T - create a tied note (with previous note)
  • L - switch note label (sharp -> flat, flat -> sharp)
  • + - raise volume by 5%
  • - - lower volume by 5%
  • Left - reduce delay
  • Right - add delay

Piano editor use similar drag & drop and resize operation for creating notes. Main difference is possibility to move/copy selected note to any other key position, thus changing the note pitch. This feature is very useful for creating chord sounds, especially with several tied notes - create tied notes in one key and then copy them with single drag & drop to other key(s).
