You can increase the quality of bass tabs by entering chord information. The given benefits are:

  • Better logical structure and understanding of the composition
  • Better visualization of notes being played on virtual fretboard in Highlight view, so you can learn playing patterns more easily. It also works without active playback - click on chord label to highlight all notes within this chord range.
  • With double-click action on chord label (in Section mode), you can trigger playback within chord range.

Chord information for current section can be assigned in Section metadata window, accessible from Main menu (). To add chord information, click on the New button and enter following You will need to set following fields:

  • Chord, composed of:
    • String - string label used for precise identification of main root note on the fretboard
    • Root note - name of the root note with optional octave number, for example Db2. When octave is not entered, first valid octave on selected string will be used.
    • Type - type of the chord, for example maj, min, ...
  • Position defining the beginning of the chord range, composed of bar, beat and subbeat index
