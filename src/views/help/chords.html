You can increase the quality of bass tabs by entering chord information. The given benefits are:

  • Better logical structure and understanding of the composition
  • Better visualization of notes being played on virtual fretboard in Highlight view, so you can learn playing patterns more easily. It also works without active playback - click on chord label to highlight all notes within this chord range.
  • With double-click action on chord label (in Section mode), you can trigger playback within chord range.

Chord information for current section can be assigned in Chord Editor, accessible from Section preferences menu. To add chord label, click on the New button and set following fields:

  • Chord, composed of:
    • Root note - name of the root note, for example: Db, C#, ...
    • Type - type of the chord, for example: maj, min, ...
    • Bass note - optional note name, used to define Slash chord - chord which indicates emphasis of a bass note other than the root of the chord.
  • Position defining start of the chord range, composed from bar, beat and subbeat index
