Main controls:

  • Use pagination buttons or mouse swipe gestures to scroll visible beats of the section
  • Use mouse wheel for zoom-out/zoom-in:
    • Scroll-up direction to raise number of visible beats on screen
    • Scroll-down direction to lower number of visible beats on screen
  • Use right mouse button to open context menu for various bar/beat operations
  • Click on the time signature label to open menu with preferences for more options
