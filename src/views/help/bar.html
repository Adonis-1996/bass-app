Toolbar controls can be divided into the tree main groups:

Track controls

Allows to select active, configure and manage instruments of the project. It also contains a menu with volume preferences of all instruments (tracks) as well as input audio channel volume when using microphone input.

Tip:   Use it to customize visual appearance of bass sound symbols

Player controls

Allows to start, stop and configure playback (speed, range, repeating)

Tip:   Use Visible screen playback or Selection playback in combination with Barline size adjustment to avoid automatic scrolling during playback for easier learning.

Section/Playlist controls

Used to switch between two application's modes:

  • Section mode - for editation or playback of single section
  • Playlist mode - for creating and playback of multiple sections - playlists

Main controls:

  • Use pagination buttons or mouse swipe gestures to scroll visible beats of the section
  • Use mouse wheel for zoom-out/zoom-in:
    • Scroll-up direction to raise number of visible beats on screen
    • Scroll-down direction to lower number of visible beats on screen
  • Use right mouse button to open context menu for various bar/beat operations
  • Click on the time signature label to open menu with preferences for more options
