Overview

  • Primarily designed for bass players
  • Exact counting labels to not get lost in more complex rhythms
  • Easy to create drums backing tracks for everyday practising
  • Clean and Intuitive user interface
  • Web version already available, desktop and mobile are planned for the future
  • More features will be coming

Playback

  • Quality playback of Bass, Drums, Piano
  • Adjustable playback speed
  • Functions for selecting playback range
  • Instruments mixer

Editor

  • Interactive Drag & Drop and resize
  • Designed separately for each specific intrument
  • Focused on simplicity, but advanced options are also available (slide timing, note bending)
  • Playlist editor allows quick reuse of repetitive sections or creation of different versions
  • Integrated projects manager for local storing of your projects

Visualization

  • Very easy to read for the beginners
  • Several bass notations to choose from
  • Adjustable layout and automatic scrolling to fit your screen and reading speed
  • Virtual fretboard synchronized with playback and chord information

Bass editor

Bass editor with opened Sound Properties context menu of selected sound. Audio signal visualization tool is enabled and can be used for time-domain analysis of selected track or bass guitar connected through microphone input.

Fretboard can be used for quick preview of sound samples, for creating sounds with drag & drop operation, or for visualization of played notes during playback.

Drums and Percussions editor

Drums/Percussions editor is much simpler. Sounds are defined only by time position and volume intensity. Drag & drop operation is also available for faster copying or moving of single sounds.

Piano editor

In the piano editor, every note (key) is represented in a separate row. Editing consist of drag & drop and resize operations (similar to the bass editor), and a few keyboard controls.

Horizontal layout with fretboard

Application in the Playlist mode with single row layout and horizontal scrolling, and with Fretboard tool enabled. Suitable for smaller screens or using in videos.

Horizontal layout

Application in the Playlist mode with 2-rows layout and vertical scrolling. Also a different notation is presented here - with fret numbers only and without colors.

Playlist editor

Playlist is a sequence of already composed sections in the project. Playlist editor also allows to easy duplicate repetitive sections.

You can define as many playlists as you wish, either for favorite parts that you often use for practising, or to create a different version of the same song.
